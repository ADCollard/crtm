CRTM Simple Test Data Files

This page is where you will find a simple dataset for use with CRTM testing. It consists of the ECMWF 52 profile set with some cloud and aerosol data tossed in. Be aware that no check has been performed on the physicality of combinations of atmospheric, cloud, or aerosol profiles so don't be surprised if you see a snow cloud associated with a warm tropical temperature profile. The intent of this simple dataset is simply for initial testing of the CRTM.

Atmosphere Data File Contents

By the way, if anyone wants to contribute some meaningful cloud profile data, please feel free to do so! The data I/O functionality in the CRTM_Cloud_Binary_IO module can also handle cloud only output. I can then intersperse this data amongst the base 52 profiles.

Download Notes

  • The files listed below in the download links section are sequential, unformatted data files (referred to as "Binary" here). They are provided in both big- and little-endian byte ordering.
  • The "ECMWF" prefix for the surface data file is simply to indicate it's a match for the similarly prefixed atmosphere data file.
