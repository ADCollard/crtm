Download Notes

  • The files listed below in the download links section are the netCDF format files. If you want to use these data in the CRTM, you can either,
    • Convert them to the expected binary format. You'll want to download the TauCoeff_NC2BIN tarball for that.
    or
    • Go directly to my anonymous ftp download site for the Binary format files. See the Download Links section for the ftp site links.

Download Links

Anonymous ftp site links

If you need to download lots of TauCoeff data files, rather than click on all the individual file links shown below in the datafile lists, just visit my anonymous ftp sites:
  • netCDF format; ftp://ftp.ssec.wisc.edu/paulv/CRTM/Coefficient_Data/netCDF/TauCoeff
  • Big-endian Binary format; ftp://ftp.ssec.wisc.edu/paulv/CRTM/Coefficient_Data/Big_Endian/TauCoeff
  • Little-endian Binary format; ftp://ftp.ssec.wisc.edu/paulv/CRTM/Coefficient_Data/Little_Endian/TauCoeff
