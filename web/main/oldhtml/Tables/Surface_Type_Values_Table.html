Surface%Type Valid Values
Surface Type Parameter Name
Invalid INVALID_SURFACE
Land LAND_SURFACE
Water WATER_SURFACE
Snow SNOW_SURFACE
Ice ICE_SURFACE

Apart from the INVALID_SURFACE parameter, the other surface types are additive so a user can specify combinations of the recognized surfaces. Thus if one sets, 

  Surface%Type = LAND_SURFACE + SNOW_SURFACE

this indicates the surface type is a combination of land and snow surface. Combinations of all the valid surfaces are supported (even if unlikely to occur in practice.)
